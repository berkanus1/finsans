# Finsans

A simple personal budget tracker to manage income, expenses, and monthly summaries.

## Features

- Log income and expense transactions
- Categorize spending
- View balance at any time

## Requirements

- Python 3.8+

## Getting Started

```bash
git clone https://github.com/berkanus1/finsans.git
cd finsans
pip install -r requirements.txt
```

## Usage

**Web arayüzü (tarayıcıdan):**
```bash
python app.py
```
Tarayıcıda `http://localhost:5000` adresini aç.

**Komut satırı:**
```bash
python budget.py
```
